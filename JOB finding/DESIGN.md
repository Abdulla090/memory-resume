# Design System: JOB Finding

## Brand Direction
Create a product that feels like a serious, modern job search tool with SubQ-level polish:
- precise
- technical
- calm
- trustworthy
- mobile-safe

## Visual Language
- Background: soft white or fog gray
- Surface: pure white cards
- Text: dark ink, not pure black
- Accent: one blue or teal system only
- Borders: thin, soft, low-contrast
- Shadows: gentle and wide, never heavy

## Typography
- Use a modern sans family only
- Headlines should be bold and clear
- Body text should stay highly readable
- Numeric data can use monospace for precision

## Layout
- Use a strong centered content container
- Keep search/filter/results easy to scan
- Prefer two-column desktop layouts and one-column mobile layouts
- Make results list dense enough to be useful, not cramped

## Components
- Search bar: large, obvious, dominant
- Filter chips: rounded, compact, clear active state
- Job cards: white cards with title, company, location, salary, tags, and CTA
- Detail panel: structured sections with clear headings
- Saved state: subtle badge, not loud

## Interaction
- Hover should be subtle
- Active states should be obvious
- Loading should use skeletons, not spinners
- No noisy animations

## Mobile Rules
- Everything must collapse cleanly
- Filter controls should be accessible from the thumb zone
- Cards should stack with enough spacing
- Touch targets must be large
- No horizontal scrolling

## Banned Patterns
- Purple AI gradients
- Random decorative blobs
- Over-animated dashboards
- Tiny buttons
- Cluttered navigation
- Generic SaaS template look
